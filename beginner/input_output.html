Python Input and Output

Understanding Input and Output

Input and output (I/O) operations are fundamental to any program. They allow us to interact with users, read data, and display results.

Basic Input and Output

# Basic input and output
name = input("What's your name? ")  # Get user input
print(f"Hello, {name}!")           # Display output

age = input("How old are you? ")
print(f"In 5 years, you'll be {int(age) + 5} years old")

Output Formatting

      
# Different ways to format output
name = "Alice"
age = 25
height = 1.75

# Using f-strings
print(f"{name} is {age} years old")

# Using .format()
print("{} is {} years old".format(name, age))

# Formatting numbers
print(f"Height: {height:.2f} meters")  # Two decimal places
